# Card Coverage

This report tracks how card behavior is implemented. The target state is data-first card behavior through typed `CardEffectSpec` / `GameEffectSpec` entries, with bespoke handlers reserved for effects that cannot yet be expressed by shared primitives.

## Current Status

The playable milestone has no known current-catalog card-effect blockers. Current checks show no Imperium Agent cards without typed Agent behavior, no Plot or Combat Intrigues without typed triggers, and no Reserve or Imperium printed reveal placeholders. The Spice Must Flow is intentionally covered through acquire effects and a zero reveal. The follow-up sections below are coverage and extensibility notes, not required manual steps for normal current play.

## Data-Driven Non-Card Islands

- Conflict rewards are data-driven through `conflict-reward-data.ts`.
- Board spaces use structured costs, gains, influence, troop, spy, contract, combat, Maker, Sietch Tabr, and team routing fields.
- Catalog imports normalize runtime `traits` to actual faction card traits; reward, cost, icon, and deck-label attributes remain available for summaries/import metadata but do not participate in trait-gated rule checks.
